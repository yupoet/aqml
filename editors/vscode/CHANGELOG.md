# Change Log

All notable changes to the AQML VS Code extension will be documented in this file.

## [0.3.0] - 2026-03-13

### Added
- Local AQML schema bundle wired through the Red Hat YAML extension
- `vsce` packaging scripts and reproducible `package-lock.json`
- GitHub Actions workflow for VSIX packaging and release publishing
- Release guide for Marketplace publishing and manual VSIX installs

### Changed
- Switched language file icons to bundled SVG assets
- Updated installation docs to prefer VSIX packaging over copying the folder by hand

## [0.2.0] - 2026-03-13

### Added
- AQML language definition, TextMate grammar, and editor snippets
